## 项目概述

本项目是一款专为幼儿园设计的综合管理系统，通过后台管理和小程序端实现园所的高效运行与管理。系统支持监控查看、宝宝状态事件管理、嘱托叮嘱等核心功能，适用于不同阶段的幼儿园、托育中心、舞蹈培训机构以及其他儿童活动场所，助力其实现信息化、系统化管理。
